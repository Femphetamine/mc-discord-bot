# MC Ylläpito botti
Itse tehty ensimmäinen discord botti, Pythonilla kirjotettu.

Koodissa voi olla monta turhaa kohtaa, ehdota korjauksia!

# Suunnitelmat
- Automatisoida serverin setuppi, esim jos serveri häkätään, voidaan nopeasti tuottaa uusi. 
- Automatisoi päivityksiä ylläpidon kanaville esim, #säännöt, #tiedotteet, #hierarkia ja #radio. Ei tarvitse manuaalisesti omalla tilillä päivittää tekstiä, vaan voit kirjottaa päivitykset ylläpidon salaisissa kanavoissa. Botti poistaa/muokkaa vanhempia tekstejä.
- Botti "modiksi", varmistaa että roolit ovat oikeilla henkilöillä. Ja että upseeristolla voivat käyttää komentoja kanavoissa eikä etsiä itse rauhan pilaavia henkilöitä.
- Jos jäsen erotetaan, botti lähettää henkilölle yksitysviestin että he ovat erotettu MC:stä, syy ei tarvii olla?
