# MC Ylläpitobotti

Ensimmäinen itse tekemäni Pythonilla kirjoitettu Discord-botti GTA StateRP -moottorikerhoa varten.

Botti on suunniteltu helpottamaan kerhon hallintaa, ylläpitoa ja automatisointia Discordissa. Projekti on edelleen kehitysvaiheessa, joten koodissa voi olla turhia tai epäoptimaalisia ratkaisuja. Kaikki parannusehdotukset ja palaute ovat tervetulleita!

## Valmiit ominaisuudet

- Serveripohjan automaattinen luonti yhdellä komennolla
- Automaattinen roolien jako oikeille henkilöille

## Tulevat ominaisuudet

- Automatisoida serverin setup ja palautusprosessi  
  → esimerkiksi tilanteissa, joissa serveri täytyy rakentaa nopeasti uudelleen.

- Automatisoida ylläpitokanavien päivitykset  
  → kuten `#säännöt`, `#tiedotteet`, `#hierarkia` ja `#radio`.

- Mahdollistaa päivitysten kirjoittaminen salaisiin ylläpitokanaviin  
  → botti päivittää automaattisesti näkyvät viestit ja poistaa tai muokkaa vanhoja ilmoituksia.

- Toimia ylläpidon/moderaation apuna  
  → tarkistaa roolit, oikeudet ja varmistaa, että oikeilla henkilöillä on tarvittavat käyttöoikeudet.

- Lähettää automaattisia yksityisviestejä jäsenille  
  → esimerkiksi erotustilanteissa tai muissa tärkeissä ilmoituksissa.

- Lisätä visuaalisia ominaisuuksia  
  → kuten botin oman RPC/Status-järjestelmän ja vaihtuvat aktiviteetit.

## Huomio

Tämä projekti toimii samalla myös oppimisprojektina Pythonin ja Discord-bottien kehityksessä.

Botti auttaa pitämään kerhon Discordin järjestyksessä ilman jatkuvaa manuaalista ylläpitoa.

Palautetta, korjauksia ja kehitysideoita arvostetaan paljon 🙌
